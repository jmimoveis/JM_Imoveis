Configurações do GitHub

Para salvar os dados automaticamente, insira os dados do seu repositório e o Personal Access Token.

Como configurar o token
  1. Crie um token em GitHub > Fine-grained tokens.
  2. Em Repository access, selecione este repositório.
  3. Em Repository permissions, marque Contents como Read and write.
  4. Se o repositório for de uma organização, confirme se o token foi aprovado e se o dono selecionado é a organização correta.
